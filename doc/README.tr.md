# TRON Adres Üretici (Türkçe)

Go dilinde yazılmış bir TRON adres üretim aracı.

## İndirme

Dosyaları GitHub Releases sayfasından indirin.

## Kullanım

Windows:
```cmd
tronvanity-windows-amd64.exe --prefix T888
```

macOS:
```bash
chmod +x tronvanity-darwin-amd64
./tronvanity-darwin-amd64 --prefix T888
```

Linux:
```bash
chmod +x tronvanity-linux-amd64
./tronvanity-linux-amd64 --prefix T888
```
