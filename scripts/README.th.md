# สคริปต์ช่วยใช้งานสำหรับ TRON Vanity Address Generator

สคริปต์เหล่านี้ช่วยให้ผู้ใช้ที่ไม่ถนัดคำสั่ง (command line) สามารถสร้างที่อยู่ TRON แบบ vanity ได้อย่างง่ายดาย **โดยไม่ต้องพิมพ์คำสั่งเอง** ในเทอร์มินัลหรือ Command Prompt

## สคริปต์สำหรับ Windows

### วิธีใช้งาน

1. ดาวน์โหลดไฟล์ `tronvanity-windows-amd64.exe` จากหน้า [GitHub Releases](https://github.com/bigbigha/TRON-Vanity-Address-Generator/releases)
2. ดาวน์โหลดไฟล์ `windows-run.bat` จากโฟลเดอร์ `scripts` ในรีโปนี้
3. นำไฟล์ทั้งสองไฟล์ไปไว้ใน **โฟลเดอร์เดียวกัน**
4. ดับเบิลคลิกไฟล์ `windows-run.bat` เพื่อรันสคริปต์
5. ทำตามคำแนะนำที่แสดงบนหน้าจอ แล้วกรอก:
    - คำนำหน้า (prefix) ที่ต้องการให้ที่อยู่ขึ้นต้น
    - คำต่อท้าย (suffix) ที่ต้องการให้ที่อยู่ลงท้าย
    - จำนวนที่อยู่ที่ต้องการให้โปรแกรมค้นหา

### จุดเด่น

- อินเทอร์เฟซแบบข้อความ เข้าใจง่าย
- ไม่จำเป็นต้องมีพื้นฐานคำสั่ง Command Prompt
- โปรแกรมจะค้นหาที่อยู่ต่อเนื่องไปเรื่อย ๆ จนกว่าคุณจะกด `Ctrl + C` เพื่อหยุด

## สคริปต์สำหรับ macOS / Linux

### วิธีใช้งาน

1. ดาวน์โหลดไฟล์ไบนารีที่ตรงกับระบบของคุณจากหน้า [GitHub Releases](https://github.com/bigbigha/TRON-Vanity-Address-Generator/releases)

    - `tronvanity-darwin-amd64` (สำหรับ Intel Mac)
    - `tronvanity-darwin-arm64` (สำหรับ Apple Silicon Mac)
    - `tronvanity-linux-amd64` (สำหรับ Linux)

2. ดาวน์โหลดไฟล์ `unix-run.sh` จากโฟลเดอร์ `scripts`
3. นำไฟล์ไบนารีและไฟล์ `unix-run.sh` ไปไว้ใน **โฟลเดอร์เดียวกัน**
4. เปิด Terminal แล้วเปลี่ยนไปยังโฟลเดอร์นั้น
5. ให้สิทธิ์รันสคริปต์: `chmod +x unix-run.sh`
6. รันสคริปต์: `./unix-run.sh`
7. ทำตามคำแนะนำที่แสดงบนหน้าจอ

### จุดเด่น

- ตรวจสอบระบบปฏิบัติการและสถาปัตยกรรมของเครื่องคุณให้อัตโนมัติ
- อินเทอร์เฟซแบบข้อความ ใช้งานง่าย
- ไม่ต้องมีความรู้คำสั่งขั้นสูงในเทอร์มินัล
- ช่วยตั้งสิทธิ์ให้ไฟล์ไบนารีสามารถรันได้โดยอัตโนมัติ
- โปรแกรมจะค้นหาที่อยู่ต่อเนื่องไปเรื่อย ๆ จนกว่าคุณจะกด `Ctrl + C` เพื่อหยุด

## หมายเหตุด้านความปลอดภัย

สคริปต์เหล่านี้เป็นเพียงตัวหุ้ม (wrapper) แบบง่าย ๆ ที่ใช้เรียกไฟล์ไบนารีภายในเครื่องของคุณเท่านั้น โดย **จะไม่**:

- เชื่อมต่ออินเทอร์เน็ต
- อัปโหลดหรือส่งข้อมูลใด ๆ ออกไป
- ติดตั้งซอฟต์แวร์เพิ่มเติม
- แก้ไขไฟล์ระบบของเครื่อง

ทั้งสคริปต์และโปรแกรมหลักจะทำงานแบบ **ออฟไลน์ 100%** บนคอมพิวเตอร์ของคุณเอง
