# Вспомогательные скрипты для TRON Vanity Address Generator

Эти скрипты предоставляют простой интерфейс для нетехнических пользователей, позволяя генерировать TRON vanity‑адреса **без прямой работы с командной строкой**.

## Скрипт для Windows

### Использование

1. Скачайте `tronvanity-windows-amd64.exe` со страницы [GitHub Releases](https://github.com/bigbigha/TRON-Vanity-Address-Generator/releases).
2. Скачайте файл `windows-run.bat` из этой папки (`scripts`).
3. Поместите оба файла в **одну и ту же папку**.
4. Дважды щёлкните по `windows-run.bat`, чтобы запустить скрипт.
5. Следуйте инструкциям на экране: введите желаемый префикс (prefix), суффикс (suffix) и количество генерируемых адресов.

### Особенности

- Простой текстовый интерфейс.
- Не требуется знание командной строки.
- Скрипт будет повторять генерацию, пока вы не нажмёте `Ctrl + C` для выхода.

## Скрипт для macOS / Linux

### Использование

1. Скачайте подходящий бинарный файл для вашей системы со страницы [GitHub Releases](https://github.com/bigbigha/TRON-Vanity-Address-Generator/releases):
    - `tronvanity-darwin-amd64` (Intel Mac)
    - `tronvanity-darwin-arm64` (Apple Silicon Mac)
    - `tronvanity-linux-amd64` (Linux)
2. Скачайте файл `unix-run.sh` из этой папки (`scripts`).
3. Поместите бинарный файл и `unix-run.sh` в **одну и ту же папку**.
4. Откройте терминал и перейдите в эту папку.
5. Выдайте права на исполнение скрипта: `chmod +x unix-run.sh`
6. Запустите скрипт: `./unix-run.sh`
7. Следуйте инструкциям на экране.

### Особенности

- Автоматически определяет вашу операционную систему и архитектуру.
- Простой текстовый интерфейс.
- Не требует продвинутых знаний командной строки.
- Автоматически делает основной бинарный файл исполняемым.
- Скрипт будет повторять генерацию, пока вы не нажмёте `Ctrl + C` для выхода.

## Заметка по безопасности

Эти скрипты — это простые оболочки (wrappers), которые только вызывают локальный бинарный файл. Они **не**:

- обращаются к интернету;
- загружают или отправляют какие‑либо данные;
- устанавливают программное обеспечение;
- изменяют системные файлы.

И скрипты, и основной бинарный файл работают полностью **офлайн** на вашем локальном компьютере.
